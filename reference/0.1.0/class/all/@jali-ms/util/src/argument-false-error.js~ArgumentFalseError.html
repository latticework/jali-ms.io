    ArgumentFalseError

    Extends:

    Error → ArgumentError → ArgumentFalsyError → ArgumentFalseError

    Represents that an argument erroneously has a value of false.

    Throw this Error if a parameter must be true.

    See:

    • package @jali-ms/util
    • module @jali-ms/util/errors
    • verifyTrue
    • verifyTruthy

    Example:

    The argument for the parameter isValid is `false`.
    throw new ArgumentFalseError('lastName');
